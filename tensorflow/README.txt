Description of the folder contents

XOR.py 
Implementation of a 2-layers neural network to solve the XOR problem.
It includes: definition of the dataset, definition and training of the model, plot separation surfaces.

mnist_mlp_essential.py
Minimal implementation of a 2-layers neural network to solve the MNIST problem.
It only includes: download and loading of the dataset, definition and training of the model, plot the gradient.

mnist_mlp.py
Implementation of a 2-layers neural network to solve the MNIST problem.
It includes: download and loading of the dataset, definition and training of the model, regularization criterion and validation check, plot the gradient.

mnist_cnn.py 
Implementation of a convolutional neural network to solve the MNIST problem.
It includes: download and loading of the dataset, definition and training of the model, plot the filters.
